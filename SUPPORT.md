# Support

Need help? Here is where to go:

- 📖 **Documentation:** [mcpdesc.org](https://mcpdesc.org)
- 🧪 **Try the editor:** [editor.mcpdesc.org](https://editor.mcpdesc.org)
- 💬 **Questions & ideas:** join
  [Discussions](https://github.com/orgs/mcpdesc/discussions) or an issue in the
  relevant repository.
- 🐛 **Bugs:** open an issue in the affected repository using the bug template.
- 🔒 **Security issues:** follow [SECURITY.md](SECURITY.md) — do **not** open a
  public issue.

Please search existing issues and discussions before opening a new one.
